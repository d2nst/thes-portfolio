새로움을 찾아다니는
모험가 김경진 입니다.

Introduce My Self

Kim Kyoung Jin 28 Feb 1995
Github Mail

머릿속 아이디어가 화면에 구현될 때의 성취감이 저를 퍼블리셔의 길로 이끌었습니다. 사용자에게 최고의 경험을 선사하기 위해 디자인을 코드로 완성하며, 디테일한 부분까지 놓치지 않고 최선의 결과를 추구합니다.

늘 새로움을 탐험하며, 즐거운 아이디어를 발견하고, 더 나은 경험을 만들기 위해 한 걸음 더 나아갑니다. 트렌드를 놓치지 않고 발 빠르게 적용하며, 문제를 해결하기 위해 적극적으로 소통하고 협업합니다.
